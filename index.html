This site is divided into sections, can be accessed from the menu at the bottom,
each section is an assignment and inside is a page dedicated to the exercise to be done.
The svelte and sveltekit framework was used to create this site, the styling is handled
in CSS and the deployment of the application to the github pages is done using the gh-pages library and the script found within the repository, as package manager npm was used.
Subsequently everything will be deployed with a github-action.
The scripts folder in the repository also contains the various python scripts used to generate the datasets or the visualisations.
